# Doc Gardening

Docs are part of the architecture, not a side channel.

When package boundaries change:
- update ARCHITECTURE.md
- update docs/architecture/dependency-graph.md
- update package docs/index.md, docs/architecture.md, and docs/public-api.md
- update AGENTS.md only if routing changes
- add or update governance lints when a rule is enforceable

Keep AGENTS.md files short. Move details to docs.
